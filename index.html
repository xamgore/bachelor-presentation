Сопоставление с образцом

Пусть n : nat — переменная

Нужно знать определение типа nat, чтобы сделать сопоставление 

      match n with
      | O    => "one"
      | S n' => "two"
      end.

Вызывается метод компилятора для получения конструктора имён nat 11
